
# GamaGit
### :rocket: _Projeto desenvolvido durante as aulas de `React` do `Hiring Coders #2` - Gama Academy_ 


## A Aplicação:
A aplicação tem o funcionamento simples: na página inicial, o usuário insere um nome de Usuário do GitHub. A aplicação, por sua vez, busca por este usuário, recupera seus Repositórios e os exibe em outra página. Caso o usuário não seja encontrado, retorna mensagem de erro.
